## Sobre o Projeto

    Esse foi um desafio para criar um jogo de memoria contruido em JavaScript.

### ▶ Desafio
    Crie um jogo de memória com cartas que possam ser viradas para revelar imagens.
    O jogador deve tentar encontrar todos os pares correspondentes.

### ⛔ Regras
    A aplicação deve ser feita apenas com Javascript sem Utilização de Frameworks.
    O jogo deverá conter imagens em formato de cartas e ao clicar na carta a mesma deverá ser virada.
   

### ✨ Descrição
    O projeto começa com uma introdução ao JavaScript e HTML básicos para criar
    uma estrutura básica de página web. Em seguida, criei um array com imagens
    e uma funções JavaScript para gerar imagens aleatórias e manipular o HTML
    para exibir as cartas do jogo.
    
    Depois da estrutura básica do jogo, adicionei interatividade ao jogo, 
    como virar as cartas e verificar se duas cartas selecionadas são um 
    par correspondente.
