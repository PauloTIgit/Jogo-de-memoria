## Jogo de Memoria

Esse e um jogo de memoria contruido em JavaScript, paseado do desaafio do discord do servidor "Mais um DEV".

### ▶ Construa um jogo de memória
    Crie um jogo de memória com cartas que possam ser viradas para revelar imagens. O jogador deve tentar encontrar todos os pares correspondentes.

### ⛔ Regras
    A aplicação deve ser feita apenas com Javascript sem Utilização de Frameworks.
    O jogo deverá conter imagens em formato de cartas e ao clicar na carta a mesma deverá ser virada.
